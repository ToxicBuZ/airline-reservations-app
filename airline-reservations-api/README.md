# Airline Reservations Backend

This project was generated with Express.

Navigate to the airline-reservations-api directory then use {{ npm install }} to install dependencies.

## To run the project use the commands below.

- On Mac OS or Linux -> DEBUG=myapp:\* npm start
- On Windows Command Prompt -> set DEBUG=myapp:\* & npm start
- On Windows Powershell -> $env:DEBUG='myapp:\*'; npm start

Then load http://localhost:3000/ in your browser to access the app.
